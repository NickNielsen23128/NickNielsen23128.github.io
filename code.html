HOW IT WAS DEVELOPED

This page goes into how the project was developed and the process I went through. Click the link to my repository below and download the code to follow along.




To start off the design procces I drew out all of the pages onto a peice of paper in order to visualize the layout. As you can see, the real website ended up being vastly different than the mock up.


I edit all of my code using Visual Studio code on a Mac. In order to veiw my projects and edit in real time, I use the extension live server for Visual Studio code.



The first step that I took when designing to project was making the JS. I will be mainly going over the development of the JS in this page. To start, I needed a way to randomize word selections of different lengths. I added an external JS script with an array filled with my word list. I imported this list and designed a random word selector from this array. Below is a picture using the console to print a randomly chosen word.




The next part of the development was the creation of the frontendle boxes where the letters would go. I tried two different approaches. The first approach was using a for loop to create thirty different divs using the createElement() function and by assigning each a unique id one through thirty. This way I could easily navigate by using the function getElementById(i) to iterate through the divs.



The second approach entailed creating divs rows and div boxes insie each row using the appendChild() function. I decided to use this as I beleived it would make it easier to make rows of different lengths as well as to have the abilit to compare individual rows to the correct string. Here is an example of the div classes set up.




A little CSS to each of the divs as well as aligning the boxes in colums with flexbox made the process a lot easier to visualize.




Next I used a event listener with the paramater keyup in order to parse the key function into a string and insert each string into each respective box using the textContent() function.





The final major step of the JS was getting the boxes to change colors based on if the string was correct. I changed both strings into arrays where I used the indexOf() function to determine what color the box should be. I changed the color to green if the indexOf() was the same for the correct guess and the user guess. If the indexOf() returned a negative number, the letter was not in the word at all and I returned a grey box. If the indexOf() returned a positive number the letter was not in the correct place and I returned a yellow box.




Not as important but equally as intresting is adding animations. I needed each box to animate at a diffrent time in order which proved to be a challenge. I ended up using the setTimeout(() => {}) function in order to set a delay. I then used a for loop inside the function to change the length of the delay each time. By using the website Animista and the function ClassList.add() I was able to set animations to each individual box.
