ABOUT

Hello, my name is Nick Nielsen and I am the designer of this website. Frontendle is an adaptation of the New York Times Wordle using the GitHub color scheme. I was initally given this assignment for my Web Page Coding class and have since published it on my GitHub website.


Browese this website using the navigation menu at the top. Feel free to inspect element the page to veiw the console as well as download and minipulate the code to learn about how this simple web game works. Of course, make sure to play the game and expand your frontend vocabulary.
